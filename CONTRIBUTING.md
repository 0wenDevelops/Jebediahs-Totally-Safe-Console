# Contributing

Thanks for wanting to improve **Jebediah's Totally Safe Console**.

This is a personal source-available project, not an open-source free-for-all. Small fixes, bug reports, and suggestions are welcome, but redistribution, reuploads, public forks, rebrands, and modified public releases are not allowed by the license.

## Good Contributions

Good contributions include:

- Bug reports with clear steps to reproduce
- Crash reports with logs
- Compatibility notes for different KSP versions
- Documentation fixes
- Small code fixes
- Feature suggestions that fit the project

## Before Opening an Issue

Check that:

- You installed the mod inside `GameData`
- The DLL is inside the mod's `Plugins` folder
- You restarted KSP after installing
- You tested with the latest available version
- You can explain what you expected to happen and what actually happened

## Bug Report Checklist

Include:

- KSP version
- Mod version or commit
- Operating system
- Other installed mods
- What scene you were in, such as Space Center, VAB, SPH, flight, map view, or tracking station
- Steps to reproduce
- Screenshots or logs, when useful

## Pull Requests

Pull requests should be small and focused.

Good pull requests:

- Fix one issue at a time
- Keep code readable
- Avoid unrelated formatting changes
- Do not include built DLL files unless requested
- Explain what changed and why
