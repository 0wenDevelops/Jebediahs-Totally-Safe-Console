# Security Policy

## Supported Versions

Only the latest public release is supported.

Older builds may still work, but fixes should target the newest version unless there is a strong reason not to.

## Reporting Security Issues

Do not post security-sensitive issues publicly if they include:

- A way to damage saves automatically
- A way to execute unwanted code
- A malicious install method
- Suspicious binaries
- Fake downloads or mirrors
- Anything that could trick users into installing unsafe files

Instead, contact the project owner privately through GitHub or another trusted channel.

## Safe Download Rule

Only download this project from the official repository or official release links.

Do not trust random mirrors, reuploads, Discord attachments, third-party installers, or modified builds claiming to be official.

## Mod Safety Notes

This project is intended for single-player KSP use. Always back up saves before testing experimental features.
